WorldBankDataProvider Type

Namespace: FSharp.Data
Assembly: FSharp.Data.dll
This is a provided type definition

Typed representation of WorldBank data with additional configuration parameters. See http://www.worldbank.org for terms and conditions.

Static parameters

Static parameters Description

Asynchronous: bool(optional,default=false)

Full Usage: Asynchronous: bool(optional,default=false)

Generate asynchronous calls. Defaults to false.

Sources: string(optional,default="WorldDevelopmentIndicators;GlobalFinancialDevelopment")

Full Usage: Sources: string(optional,default="WorldDevelopmentIndicators;GlobalFinancialDevelopment") 

The World Bank data sources to include, separated by semicolons. Defaults to World Development Indicators;Global Financial Development.
                        If an empty string is specified, includes all data sources.
